## 🚀 Troubleshooting typical errors

If you see an error code during setup, verify your internet connection. A weak signal cuts off the download mid-way and results in a corrupt file. If the problem persists, delete the file you downloaded earlier, clear your browser cache, and download the installer again. 

Should the software fail to open, check if your antivirus blocks the application. Some security programs mistake new software for a threat. Add an exception for the AMD Radeon directory in your antivirus settings.

If your game performance drops after an update, roll back the driver. You can do this through the Device Manager in Windows. Find your display adapter, right-click it, select properties, and choose the Roll Back Driver option. This puts your card back to the previous stable state while you wait for a new patch.
